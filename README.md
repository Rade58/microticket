KREIRACU NOVI NEXTJS PROJEKAT, KOJI KORISTI TYPESCRIPT

- `mkdir client`

- `cd client`

- `yarn init -y`

- `yarn add next react react-dom`

- `touch tsconfig.json`

- `yarn add typescript @types/react @types/node --dev`

- `touch next-env.d.ts`

- `code package.json`

```json
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
}
```

- `cd ..`

- `code .gitignore`
